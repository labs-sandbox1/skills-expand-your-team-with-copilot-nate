# How to Request Changes

## For Teachers and Staff

If you need to make changes to the Activities website, you don't need to know how to code! We've created easy-to-use forms to help you submit your requests clearly.

## Opening an Issue

1. Go to the [Issues tab](../../issues) in GitHub
2. Click the green **"New Issue"** button
3. Choose the form that matches what you need

## Tips for Good Requests

### Be Specific
Instead of: "Fix the activities"
Write: "The Chess Club meeting time shows 3:00 PM but should be 3:30 PM"

### Explain Why
Include context about why the change is needed:
- "Students are confused because..."
- "This information is outdated..."
- "Parents have been asking for..."

### Include Examples
If you're describing a new feature, explain how similar features work on other websites you've seen.

## What Happens Next?

1. **You submit the form** - The issue is created with all your details
2. **We review it** - Staff will look at your request and may ask clarifying questions
3. **Work begins** - The request will be assigned to GitHub Copilot or a developer
4. **You're notified** - You'll receive updates when work is completed

## Need Help?

- Not sure which form to use? Pick the one that seems closest and explain your needs in the description
- Have multiple related changes? You can list them all in one form
- Have questions? Start a [Discussion](../../discussions) instead
